# Thankful-site
For one and everyone 
Как поднять сайт локально: 
1. Скачать папку dist
2. Клик правой кнопкой мыши по папке - Новый терминал по адресу папки <img width="303" height="478" alt="image" src="https://github.com/user-attachments/assets/2ba8f6b5-8321-41c5-8750-92449b15ca1a" />
3. В терминале ввести: python3 -m http.server 8000 ,где 8000 - порт (может быть любой целочисленный натуральный).
4. В брузере зайти на свой хост, в нашем случае: localhost:8000
5. Сайт запущен для локального просмотра.
